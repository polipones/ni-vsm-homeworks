# VSM homeworks 2021 (frycdomi, hojektom, pickaja3)

## Prepare projects

### Linux

1. Create virtual environment

```bash
python3.9 -m venv venv
```

2. Activate venv

```bash
source venv/bin/activate
```

3. Install requirements

```bash
pip install -r requirements.txt
```
